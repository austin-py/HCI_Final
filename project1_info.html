This looping animation was created for my 396-Generative Methods class using p5 and JS. This class explores the generative abilities of computers using html, css, and JS. Other projects for the class have included creating reactive masks, a chatbot, and data visualizations.

I aimed to create an almost perfect loop where the user could almost not tell when the loop restarts. I would say I was mostly succesful, although there is a slight jump when the animation restarts due to browser lag most of the time.

I really enjoy this animation and how it gradually changes colors throughout. I find the shape it draws quite satisfying as I have always enjoyed shapes created from Sin and Cosine curves.
