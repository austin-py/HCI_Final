This particle animation was created for my 396-Generative Methods class using p5 and JS. This class explores the generative abilities of computers using html, css, and JS. Other projects for the class have included creating reactive masks, a chatbot, and data visualizations.

My main goal for this project was to create a visually intersting system of particles. I did this by having them randomly generated on the page, and having an attraction force towards the center. Once they get to the center they are repelled back. Over time these effects stack and create an unpredictable and super satisfying to watch swarm of white circles.

I really enjoyed learning how to create forces in p5 and how to draw them in a way that was not confusing for the viewer. This project also taught me about how the small random forces can stack over time and create an unpredictable view for each user.
